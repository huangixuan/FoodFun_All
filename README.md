# FoodFunu點餐平台
屬於自己的點餐系統

## 系統介紹
本系統為線上訂餐平台，因應餐飲業者轉型，製作獨家官網，提供店家做產品行銷，餐點特色介紹，同時結合消費者及店員的點餐及接單系統，讓店鋪管理更方便。

## 系統特色
1. 節省公司點餐人力成本。
2. 節省點餐時間，增加翻桌率。
3. 菜單更新快速，上傳雲端即時生效。
4. 即時連線隨時掌握產品庫存，方便進銷存管理。
5. 店家介紹，讓顧客深入了解店家及餐點特色
